Welcome

This site presents research and findings about web browsers, their history, modern developments, and key concepts. It was created as an IT 3203 class project.

Abstract

Web browsers are the interface between users and the World Wide Web. They receive, interpret, and display resources (HTML, CSS, JavaScript, images, video) and have evolved from simple text renderers to powerful application platforms.
